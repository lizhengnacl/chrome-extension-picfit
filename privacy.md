# PicFit 图适配 隐私权政策

最后更新日期：2026年3月10日

## 信息收集与使用

PicFit 图适配是一款本地运行的浏览器扩展程序，我们承诺：

- 不收集任何个人身份信息
- 不收集用户的浏览历史记录
- 不访问或收集用户的书签、标签页信息
- 所有图片处理操作均在用户本地浏览器中完成，图片数据不会上传到任何外部服务器

## 数据存储

本扩展仅使用 Chrome Storage API 保存用户的使用偏好设置（如图片格式、质量参数等），这些数据仅存储在用户本地设备上，不会被发送到任何第三方。

## 第三方服务

本扩展不使用任何第三方分析服务、广告服务或数据收集服务。

## 变更

我们可能会不时更新本隐私权政策。任何变更都将在本页面发布，并更新顶部的"最后更新日期"。
